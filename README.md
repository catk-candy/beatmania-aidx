# beatmania-aidx

Web Audio APIとCanvas APIを利用した、ブラウザ上で動作する8キー対応の音楽ゲーム（リズムゲーム）です。付随する `chartGenerator.py` を使用することで、任意のWAVファイルから自動で譜面（JSON）を生成することができます。

## ゲームの仕様 / プレイスタイル

### 操作キー (8 Keys)
ゲームは以下の8つのキーを使用してプレイします。
- **白・赤鍵盤 (フルサイズ)**: `Shift`, `Z`, `X`, `C`, `V`
- **オレンジ鍵盤 (ナローサイズ)**: `S`, `D`, `F`

### リザルト・判定システム
楽曲の再生が最後まで終了すると、リザルト画面が表示されます。
スコアは **「判定スコア合計 / (全ノーツ数 × 1000)」** の達成率に基づいてランク付けされます。
- **AAA**: 8/9 以上
- **AA**: 7/9 以上
- **A**: 6/9 以上
- **B**: 5/9 以上
- **C**: 4/9 以上
- **D**: 3/9 以上
- **E**: 2/9 以上
- **F**: 2/9 未満

## 譜面生成スクリプト (`songs/chartGenerator.py`)

LibROSAを用いて楽曲のメロディ、パーカッション、オンセット強度を検出し、楽曲の展開（Drop, Build-up, Breakなど）に合わせた譜面を自動生成します。音声データ等の素材とスクリプトはすべて `songs/` ディレクトリ内で管理されます。

### 特徴的なロジック
- **難易度スケーリング**: 引数に渡したレベル (`1〜12`) に応じて、目標ノーツ数（`レベル × 100`）が設定されます。
- **低難易度向け調整 (Lv1〜Lv5)**: 曲の終盤（進行度70%以降）のノーツ密度を意図的に下げる「アンチスパイク（ラス殺し防止）」機能が働きます。
- **初心者向けレーン制限 (Lv1〜Lv3)**: Lv1〜3の譜面を生成する場合、**オレンジ鍵盤（S, D, F）には一切ノーツが降ってこない** ように制限がかかり、白・赤鍵盤のみでプレイできるようになります。

### 実行環境の準備
```bash
pip install librosa numpy scipy
```

### スクリプトの実行方法
`songs` ディレクトリ内で以下のコマンド形式で実行します。

```bash
cd songs
python chartGenerator.py [入力ファイル名] [譜面レベル(1-12)]
```

#### 引数の詳細
1. **入力ファイル名（省略可）**:
   - 対象とする音声ファイル名（拡張子 `.wav` は省略可能）を指定します。
   - デフォルトは `song` (`song.wav` を探します)。
2. **譜面レベル（1〜12）**:
   - 1〜12の整数で指定します（範囲外は自動でクリップされます）。
   - デフォルトは `1` です。

#### 実行例
ディレクトリ内にある `my_music.wav` を読み込み、難易度レベルが8の譜面データを生成します。
```bash
python chartGenerator.py my_music 8
```
- 出力先: `my_music_08.chart.json`

生成後は、`songs/songs.json` のリストの配列に出力された譜面のファイル名・タイトル・音源情報を追記することで、ゲーム画面の曲リストに反映されます。

## ゲームの起動方法
ブラウザからアクセスするために、ルートディレクトリでローカルサーバーを起動します。

```bash
python server.py
```
デフォルトでは `http://localhost:8000` でサーバーが立ち上がります。ブラウザでアクセスしてゲームをプレイしてください。
